# SDC355-JavaScript.Project
Personal Portfolio Website

Overview:
  This project is a personal portfolio webpage created to demonstrate front-end development concepts using HTML, CSS, and JavaScript, with an emphasis on interactivity, dynamic content, and user experience.

The portfolio includes:
  A custom welcome modal displayed on page load
  A dark mode slider with saved user preferences
  Dynamically rendered projects using JavaScript and sessionStorage
  Conditional featured content
  A contact form with tooltips and timed confirmation feedback
This project is fully client-side and does not require a server to run.

Languages & Technologies Used
  HTML – Page structure and semantic layout
  CSS – Styling, layout, dark mode, modal overlays, and tooltips
  JavaScript – DOM manipulation, events, conditionals, loops, storage APIs

Libraries & Frameworks
  This project does not use any external libraries or frameworks.
  All functionality is implemented using vanilla JavaScript and standard web APIs.

Key Features:
  Custom Welcome Modal (replaces prompt())
  Dark Mode Toggle with persistence using localStorage
  Dynamic Project Rendering using JavaScript objects and sessionStorage
  Conditional Logic to display featured content
  Skill List Rendering using loops
  Interactive Contact Form with:
    Tooltip on submit button
    Prevented page refresh
    Timed confirmation message

How to Run the Project:
  Download or clone the project files
  Ensure all files are in the same directory
  Open index.html in a web browser
No build steps or installations are required.

Storage Usage:
  localStorage: Saves the user’s dark mode preference
  sessionStorage: Stores and retrieves project data dynamically
